标签的属性顺序

HTML 属性应当按照以下给出的顺序依次排列，确保代码的易读性。 （1）class （2）id, name （3）data-* （4）src, for, type, href, value （5）title, alt （6）role, aria-* role的作用是描述一个非标准的tag的实际作用。比如用div做button，那么设置div 的 role="button"，辅助工具就可以认出这实际上是个button。 aria的意思是Accessible Rich Internet Application，aria-*的作用就是描述这个tag在可视化的情境中的具体信息。
