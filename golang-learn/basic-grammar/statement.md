## 标记

```go
// Go 程序可以由多个标记组成，可以是关键字，标识符，常量，字符串，符号。如以下 GO 语句由 6 个标记组成：

// fmt.Println("Hello, World!")

// 6 个标记是(每行一个)：

1. fmt
2. .
3. Println
4. (
5. "Hello, World!"
6. )
```

## 行分隔符

- 在 Go 程序中，一行代表一个语句结束。
- 每个语句不需要像 C 家族中的其它语言一样以分号 ; 结尾，因为这些工作都将由 Go 编译器自动完成。
- 如果你打算将多个语句写在同一行，它们则必须使用 ;
