## 使用 markmap 生成 md 文档的思维导图

输出的是一个 html 文件。

`markmap BLOG-markdown语法学习.zh-CN.md --no-open -o ./_markmap/markdown语法学习.html`
`markmap README.md --no-open -o ./_markmap/README.html`
