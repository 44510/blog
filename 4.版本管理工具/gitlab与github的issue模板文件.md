# gitlab 与 github 的 issue 模板文件

## gitlab

创建`.gitlab/issue_templates`文件夹并添加 md 模板文件。

```md
# feature

## 你期望的 API | 功能 是怎样的

描述一下你期望这个新功能的 API | 功能 是如何使用的

## 这个功能解决了什么问题

请尽可能详尽地说明这个需求的用例和场景
```
