```URL特殊符号及对应的十六进制值编码：
	1. +  URL 中+号表示空格 %2B 
	2. 空格 URL中的空格可以用+号或者编码 %20 
	3. /  分隔目录和子目录 %2F  
	4. ?  分隔实际的 URL 和参数 %3F  
	5. % 指定特殊字符 %25  
	6. # 表示书签 %23  
	7. & URL 中指定的参数间的分隔符 %26  
	8. = URL 中指定参数的值 %3D 
```

# ajax数据请求为空的情况，以及内部的属性为空的情况，在使用的时候最好先验证数据是否为空。

1. 使用```data.attr&&operate```；
2. 使用if判断语句；
3. 数组，测试```arr.length==0```,判断数组是否为空；
