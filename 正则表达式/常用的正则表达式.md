### 匹配汉字
```
[\u4e00-\u9fa5]
```
### 匹配邮箱地址
```
^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$
```

### html标签

```
^<(\S*?)[^>]*>.*?</\1>|<.*? />
```
