# 后续开发规划

## 基建项目 / infrastructure-projects

包括并不限于如下的方面：

1. 测试平台；
2. 禅道部署；
3. 公司内部邮箱；
4. 私有 npm 源；
5. 私有 gitlab；
   1. gitlab 分组：
      1. business：包括前后端代码；
      2. npm-package：发布到 npm 包管理平台的包；
      3. infrastructure：基建项目，如前后端监控；
      4. template：模板项目仓库；
6. nginx 配置：
   1. api：一般是服务端口配置；
   2. f2e：前端项目；
